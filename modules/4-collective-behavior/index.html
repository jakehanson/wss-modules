Nest Site Selection

Instructions:

See how the geometry of the environment can affect the ability for a colony to reach quorum at a given nest. Parameters such as radius, aperture size, and entry rate can be changed via the toggle bars above each nest.

Once you have chosen your settings, you can toggle the state and run a simulation of the model using the other controls:

Step
Take one step in the simulation
Start
Run the simulation
Stop
Stop the simulation
Restart
Stop and clear the simulation
